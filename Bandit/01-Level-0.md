#### LEVEL 0
**Target:** 
This level requires you to connect to the Bandit game server using SSH.

**Information:**
Log in to the Bandit server via SSH on port 2220 using the username and password bandit0, then move on to Level 1.

**Execution:**

``` bash
 ssh bandit0@bandit.labs.overthewire.org -p 2220
```
